\subsection{Notes and Warnings}\label{notes-and-warnings}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \textbf{H\_PLANT = 18.17 m} is back-calculated from system data, not
  field-measured. All operating points shift if this value changes.
\item
  \textbf{1-pump scenario} produces Q\_total \textless{} VAG Qmin (4500
  m3/h). This is a known system characteristic, not a modeling error.
\item
  Valve headloss uses ISO 5167 Kv method: dH = Q\^{}2 x 132.15 /
  Kv\^{}2. VAG's internal zeta values are \textbf{not} used.
\item
  Pump efficiency is fixed at the duty-point value for energy
  calculations. Actual efficiency varies with operating point.
\item
  EPS rule-based controls handle standby pump activation (PUMP\_5) and
  RIKO valve position changes automatically per Georgescu et al.~(CCWI
  2015).
\end{enumerate}
